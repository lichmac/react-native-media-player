# react-native-media-player

React-native Android/iOS plugin for MediaPlayer SDK

## Installation

```sh
npm install react-native-media-player
```

## Usage

```js
import { multiply } from "react-native-media-player";

// ...

const result = await multiply(3, 7);
```
